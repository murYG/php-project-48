# Brain-games

Test project "Gendiff" for Hexlet "php" course

## Setup
Setup SSH before clone:

git clone git@github.com:murYG/php-project-48.git
cd php-project-48

make install

## Launch
    gendiff [--format <fmt>] <firstFile> <secondFile>

## Prerequisites
    Linux, Macos, WSL
    PHP >=8.2
    Make
    Git
